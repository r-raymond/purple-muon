Description

Purple Moun Clients have `8` possible keys they can press. These are

  • accelerate (up key)
  • turn left (left key)
  • turn right (right key)
  • decelerate (down key)
  • fire 1 (space)
  • fire 2 (s)
  • fire 3 (d)
  • fire 4 (f)

These controls are send from the clients to the server 30 times a second. They packed into a Word8, where the bits signalize pressed (1) or not pressed (0) in the order mentioned above.
